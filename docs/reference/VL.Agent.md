# VL.Agent

The code that runs inside vvvv gamma, a C#
node library in the `Agent` category, built for `net8.0-windows` against the
published `VL.HDE` 2025.7.2 package and runtime-validated against vvvv gamma 7.2.

`VL.Agent.HDE.vl` is the preferred host: it runs the bridge in the editor/HDE
runtime, outside the user patch being edited.
